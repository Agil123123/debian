Debsources provide access to the source code of the Debian operating system.

You can start by browsing through the list of source packages, or searching for a particular one based on its name. Multiple versions of each source package might be available — currently, versions coming from stable, testing, unstable, and experimental are available.

Once chosen a specific version, you can browse through the source package structure to inspect individual source files. If you have Javascript enabled in your browser, source code will be syntax-highlighted and you will be able to point to individual lines; otherwise the raw file will be returned.

You can also use Debsources programmatically, or simply as a stable base to point to particular lines in a source file, possibly adding custom annotations to them. But hey, read the doc!
